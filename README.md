* 找片神器，高清党福音
* 独家功能：自动解析电影英文名（难道你不知道为什么要用英文？），BTDigg一键搜索（没有英文名则使用中文名）
* RARBG、SubDH 一键直达
* “低端影视”一键搜索
* SubHD、字幕库、伪射手 一键直达

* 建议使用暴力猴Violentmonkey安装

* BTDigg、RARBG需扶墙
* 全网最好用，不接受反驳
